# <a name="asymmetric-keys-encryption-strength"></a>Force de chiffrement des clés asymétriques
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Cette règle vérifie si les clés asymétriques ont été créées en utilisant le chiffrement 1024 bits ou un chiffrement plus fort.  
  
## <a name="best-practices-recommendations"></a>Meilleures pratiques recommandées  
 Utilisez le chiffrement RSA 1024 bits ou un chiffrement plus fort pour créer des clés asymétriques pour le chiffrement de données.
