[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Cette règle recherche des clés symétriques créées par l’utilisateur dans les bases de données master, msdb, model et tempdb.
