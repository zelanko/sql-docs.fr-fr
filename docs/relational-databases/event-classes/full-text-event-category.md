# <a name="full-text-event-category"></a>Catégorie d'événements Texte intégral
[!INCLUDE [SQL Server - ASDB](../../includes/applies-to-version/sql-asdb.md)]
  La catégorie d’événements **Texte intégral** contient des événements généraux de texte intégral.  
  
## <a name="in-this-section"></a>Dans cette section  
  
|Rubrique|Description|  
|-----------|-----------------|  
|[FT:Crawl Aborted, classe d’événements](../../relational-databases/event-classes/ft-crawl-aborted-event-class.md)|Indique qu'une analyse de texte intégral a rencontré une exception.|  
|[Classe d'événements FT:Crawl Started](../../relational-databases/event-classes/ft-crawl-started-event-class.md)|Indique qu'une analyse de texte intégral a démarré.|  
|[FT:Crawl Stopped, classe d’événements](../../relational-databases/event-classes/ft-crawl-stopped-event-class.md)|Indique qu'une analyse de texte intégral s'est arrêtée.|
