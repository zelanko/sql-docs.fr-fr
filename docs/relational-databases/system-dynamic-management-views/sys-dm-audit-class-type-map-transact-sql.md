# <a name="sysdm_audit_class_type_map-transact-sql"></a>sys.dm_audit_class_type_map (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-asdb-asdw-xxx-md](../../includes/tsql-appliesto-ss2008-asdb-asdw-xxx-md.md)]

  Retourne une table qui mappe le champ class_type dans le journal d'audit au champ class_desc dans sys.dm_audit_actions. Pour plus d’informations [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] sur l’audit, consultez [SQL Server audit &#40;moteur de base de données&#41;](../../relational-databases/security/auditing/sql-server-audit-database-engine.md).  

|Nom de la colonne|Type de données|Description|  
|-----------------|---------------|-----------------|  
|**class_type**|**Char (2)**|Type de classe de l'entité auditée. Mappe aux class_type écrites dans le journal d’audit et retournées par la fonction **get_audit_file ()** . N'accepte pas la valeur NULL.|  
|**class_type_desc**|**nvarchar (120)**|Nom de l'entité pouvant être auditée. N'accepte pas la valeur NULL.|  
|**securable_class_desc**|**nvarchar (120)**|Objet sécurisable mappé au class_type audité. Est NULL si le class_type n'est pas mappé à un objet sécurisable. Peut être lié à class_desc dans sys.dm_audit_actions.|  
  
## <a name="permissions"></a>Autorisations  
 Le principal doit avoir l’autorisation **Select** . Par défaut, cette autorisation est accordée à Public.
