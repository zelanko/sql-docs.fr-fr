# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  L’interface IBCPSession2 est une extension de IBCPSession qui fournit une fonction membre qui est une alternative à l’appel de IBCPSession:: BCPColFmt pour chaque colonne.  IBCPSession2 hérite de IBCPSession et ajoute une nouvelle méthode: [IBCPSession2:: BCPSetBulkMode](../../oledb/ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
