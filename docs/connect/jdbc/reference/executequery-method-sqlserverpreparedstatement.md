# <a name="executequery-method-sqlserverpreparedstatement"></a>executeQuery (méthode) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Exécute la requête SQL dans cette [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) objet et retourne le [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objet généré par la requête.
