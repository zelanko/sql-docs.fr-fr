# <a name="setshort-method-sqlserverpreparedstatement"></a>setShort (méthode) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Définit le paramètre désigné selon la donnée **court** valeur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public final void setShort(int index,  
                           short x)  
```  
  
#### <a name="parameters"></a>Paramètres  
 *index*  
  
 Un **int** qui indique le nombre de paramètres.  
  
 *x*  
  
 A **court** valeur.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode setShort est spécifiée par la méthode setShort dans l’interface java.sql.PreparedStatement.
