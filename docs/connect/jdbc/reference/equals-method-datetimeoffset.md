# <a name="equals-method-datetimeoffset"></a>Méthode equals (DateTimeOffset)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Compare cette **DateTimeOffset** objet vers un autre **objet**.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public boolean equals(Object o)  
```  
  
#### <a name="parameters"></a>Paramètres  
 *o*  
  
 Objet que vous souhaitez comparer à l'instance actuelle.  
  
## <a name="return-value"></a>Valeur retournée  
 Retourne la valeur true si cette **DateTimeOffset** objet représente le même point dans le temps avec le même décalage par rapport à GMT qu’un autre **DateTimeOffset**.
