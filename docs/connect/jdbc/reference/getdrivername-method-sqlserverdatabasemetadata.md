# <a name="getdrivername-method-sqlserverdatabasemetadata"></a>Méthode getDriverName (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère le nom de ce pilote JDBC.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public java.lang.String getDriverName()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 A **chaîne** qui contient le nom du pilote JDBC.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode getDriverName est spécifiée par la méthode getDriverName dans l’interface java.sql.DatabaseMetaData.
