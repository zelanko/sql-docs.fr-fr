# <a name="getlong-method-sqlservercallablestatement"></a>Méthode getLong (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’un **long** dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getLong (int)](../../../connect/jdbc/reference/getlong-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’un **long** en fonction de l’index de paramètre de langage de programmation Java.|  
|[getLong (java.lang.String)](../../../connect/jdbc/reference/getlong-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’un **long** en fonction du nom du paramètre de langage de programmation Java.|
