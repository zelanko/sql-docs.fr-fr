# Le&#231;on 1 : Navigation de base dans SQL Server Management Studio
[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] est un outil puissant et souple. Cependant, les utilisateurs novices de Microsoft Visual Studio rencontrent parfois des difficultés à accéder rapidement à la bonne fonctionnalité. Cette leçon montre comment les fenêtres s'articulent pour offrir une zone de conception facilitant le travail. À la fin de cette leçon, vous aurez appris comment supprimer les fenêtres inutiles, comment afficher rapidement des fenêtres avec les informations de votre choix et comment configurer la zone de conception pour vous aider dans vos tâches récurrentes.  
  
Cette leçon contient les rubriques suivantes :  
  
-   [Démarrage de SQL Server Management Studio (SSMS)](https://msdn.microsoft.com/library/ms166996.aspx)  
  
-   [Connexion avec le composant Serveurs inscrits et l'Explorateur d'objets](https://msdn.microsoft.com/library/ms170681.aspx)  
  
-   [Modification de la disposition d'environnement](https://msdn.microsoft.com/library/ms170069.aspx)  
  
-   [Afficher la fenêtre de requête](https://msdn.microsoft.com/library/ms169823.aspx)  
  
-   [Afficher la page Détails de l’Explorateur d’objets](https://msdn.microsoft.com/library/ms170076.aspx)  
  
-   [Sélectionner le schéma de raccourcis clavier](https://msdn.microsoft.com/library/ms167416.aspx)  
  
-   [Définir les options de démarrage](https://msdn.microsoft.com/library/ms166556.aspx) 
  
-   [Restaurer la configuration par défaut de SQL Server Management Studio](https://msdn.microsoft.com/library/ms166550.aspx)  
  
-   [Résumé : Navigation de base](https://msdn.microsoft.com/library/ms170043.aspx)
