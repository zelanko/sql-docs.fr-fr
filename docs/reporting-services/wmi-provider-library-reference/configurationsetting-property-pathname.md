# <a name="configurationsetting-property---pathname"></a>Propriété ConfigurationSetting - PathName
  Spécifie le chemin d'installation d'une instance de serveur de rapports. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Objet **String** qui représente le chemin d'accès d'installation.
