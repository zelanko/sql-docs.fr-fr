# <a name="configurationsetting-property---installationid"></a>Propriété ConfigurationSetting - InstallationID
  Cette propriété retourne un identificateur unique pour une instance de serveur de rapports spécifique.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
