# <a name="system-databases-in-parallel-data-warehouse-pdw"></a>Bases de données système en parallèle Data Warehouse (PDW)
Les bases de données système sont des bases de données que SQL Server PDW utilise pour gérer les utilisateurs, les bases de données et les données temporaires. SQL Server PDW comprend les bases de données système suivantes : Master et tempdb.  
  
## <a name="system-database-topics"></a>Rubriques relatives à la base de données système  
  
-   [Base de données Master](master-database.md)  
  
-   [Base de données tempdb](tempdb-database.md)
