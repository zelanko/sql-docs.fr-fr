# <a name="logical-operators-transact-sql"></a>Opérateurs logiques (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  Les opérateurs logiques testent le caractère vrai ou faux d'une condition. Opérateurs logiques, comme les opérateurs de comparaison, retournent un **booléenne** de type de données avec la valeur TRUE, FALSE ou UNKNOWN.  
  
|Opérateur|Signification|  
|--------------|-------------|  
|[ALL](../../t-sql/language-elements/all-transact-sql.md)|TRUE si tous les éléments d'un jeu de comparaisons sont TRUE.|  
|[AND](../../t-sql/language-elements/and-transact-sql.md)|TRUE les deux expressions booléennes sont TRUE.|  
|[N’IMPORTE QUEL](../../t-sql/language-elements/any-transact-sql.md)|TRUE si n'importe quel élément d'un jeu de comparaison est TRUE.|  
|[ENTRE](../../t-sql/language-elements/between-transact-sql.md)|TRUE si l'opérande est situé dans une certaine plage.|  
|[IL EXISTE](../../t-sql/language-elements/exists-transact-sql.md)|TRUE si une sous-requête contient des lignes.|  
|[IN](../../t-sql/language-elements/in-transact-sql.md)|TRUE si l'opérande est égal à un élément d'une liste d'expressions.|  
|[COMME](../../t-sql/language-elements/like-transact-sql.md)|TRUE si l'opérande correspond à un modèle.|  
|[NOT](../../t-sql/language-elements/not-transact-sql.md)|Inverse la valeur de tout autre opérateur booléen.|  
|[OU](../../t-sql/language-elements/or-transact-sql.md)|TRUE si l'une ou l'autre expression booléenne est TRUE.|  
|[CERTAINS](../../t-sql/language-elements/some-any-transact-sql.md)|TRUE si certains éléments d'un jeu de comparaisons sont TRUE.|
