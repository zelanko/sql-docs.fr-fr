# <a name="open-an-existing-solution"></a>Ouvrir une solution existante
Vous ne pouvez avoir qu'une seule solution ouverte à la fois dans l'Explorateur de solutions.  
  
### <a name="to-open-an-existing-solution"></a>Pour ouvrir une solution existante  
  
1.  Dans le menu **Fichier** , pointez sur **Ouvrir**, puis cliquez sur **Projet/Solution**.  
  
2.  Dans la boîte de dialogue **Ouvrir un projet** , accédez à la solution que vous souhaitez ouvrir, cliquez sur celle-ci, puis sur **Ouvrir**.
