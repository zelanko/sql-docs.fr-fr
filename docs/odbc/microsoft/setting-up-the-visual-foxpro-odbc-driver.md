# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>Configurer le pilote ODBC Visual FoxPro
Pour plus d’informations sur le téléchargement et installation du pilote ODBC Visual FoxPro, consultez [pilote ODBC Visual FoxPro](http://go.microsoft.com/fwlink/?LinkId=121318) de téléchargement MSDN.  
  
 Le programme d’installation du pilote ODBC Microsoft Visual FoxPro vous permet d’effectuer les opérations suivantes :  
  
-   Ajouter de nouveaux composants.  
  
-   Supprimer les composants installés.  
  
-   Réinstallez pour restaurer des fichiers et paramètres manquants.  
  
-   Supprimez tous les composants installés précédemment.  
  
 Après avoir installé le pilote sur votre système, le programme d’installation reconnaît les composants de pilote installé et présente des boîtes de dialogue supplémentaires qui vous permettent de modifier la configuration de votre pilote.
