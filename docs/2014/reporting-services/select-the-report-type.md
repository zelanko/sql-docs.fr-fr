# <a name="select-the-report-type"></a>Sélectionner le type de rapport
  Utilisez cette page de l'Assistant Rapport pour sélectionner un rapport tabulaire ou un rapport de matrice.  
  
## <a name="options"></a>Options  
 **Tabulaire**  
 Permet de créer un rapport qui contient un tableau. Un tableau est une région de données où les données sont organisées en colonnes et en lignes. Les tableaux comprennent un ensemble statique de colonnes, et le nombre de lignes qu'elle comporte dépend des données présentes dans le dataset.  
  
 Les pages suivantes de l'Assistant fournissent les options nécessaires à la définition du tableau.  
  
 **Matrix**  
 Permet de créer un rapport qui contient une matrice. Une matrice est une région de données où les données sont organisées en colonnes et en lignes. Une matrice fournit des fonctionnalités similaires à celles des analyses croisées et des tableaux croisés dynamiques. À la différence d'une table, qui est un ensemble statique de colonnes, une matrice peut comporter des colonnes dynamiques.  
  
 Les pages suivantes de l'Assistant fournissent les options nécessaires à la définition de la matrice.
