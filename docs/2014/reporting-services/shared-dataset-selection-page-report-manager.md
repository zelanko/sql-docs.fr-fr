# <a name="shared-dataset-selection-page-report-manager"></a>Page Sélection d'un dataset partagé (Gestionnaire de rapports)
  Utilisez la page Sélection d'un dataset partagé pour vérifier et modifier les datasets partagés qui sont actuellement associés à un rapport.  
  
 Vous ne pouvez pas utiliser cette page pour ajouter d'autres datasets partagés à un rapport ou pour supprimer des datasets partagés d'un rapport.  
  
## <a name="navigation"></a>Navigation  
 Utilisez la procédure suivante pour naviguer jusqu'à cet emplacement dans l'interface utilisateur.  
  
###### <a name="to-open-the-shared-dataset-selection-page"></a>Pour ouvrir la page Sélection d'un dataset partagé  
  
1.  Ouvrez le Gestionnaire de rapports et recherchez le rapport pour lequel vous souhaitez examiner les datasets partagés.  
  
2.  Pointez sur le rapport et cliquez sur la flèche déroulante.  
  
3.  Dans le menu déroulant, cliquez sur **Gérer**. La page des propriétés générales pour le rapport s'ouvre.  
  
4.  Sélectionnez l'onglet **Datasets partagés** .  
  
## <a name="options"></a>Options  
 **Parcourir**  
 Pour chaque nom de dataset partagé dans un rapport, vous pouvez vérifier le chemin d'accès au dossier et le nom actuels du dataset partagé cible.  
  
 Pour modifier le dataset partagé référencé par un nom de dataset, cliquez sur le bouton **Parcourir** .  
  
 Le bouton **Parcourir** permet d'ouvrir la structure des dossiers du serveur de rapports. Cliquez sur un dataset partagé pour ajouter le chemin d'accès complet dans le champ **Emplacement** .  
  
 Cliquez sur le bouton **OK** lorsque vous avez sélectionné un dataset partagé ou cliquez sur **Annuler** pour annuler la recherche d'un dataset partagé.  
  
 **Appliquer**  
 Enregistrez vos modifications.
