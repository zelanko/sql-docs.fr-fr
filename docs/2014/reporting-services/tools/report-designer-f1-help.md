# <a name="report-designer-f1-help"></a>Aide sur le Concepteur de rapports via la touche F1
  Cette section fournit une aide (F1) pour le [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] Assistants Concepteur de rapports, des vues et des boîtes de dialogue. Vous pouvez également ouvrir l'aide via la touche F1 lors de l'utilisation d'une boîte de dialogue en appuyant sur F1 ou en cliquant sur le bouton Aide affiché dans la boîte de dialogue.
