# <a name="delete-a-row-mds-add-in-for-excel"></a>Supprimer une ligne (Complément MDS pour Excel)
  Dans [!INCLUDE[ssMDSshort](../../includes/ssmdsshort-md.md)][!INCLUDE[ssMDSXLS](../../includes/ssmdsxls-md.md)], supprimez une ligne de données (membre) de la base de données de référentiel MDS lorsque vous ne souhaitez plus qu’elle soit à la disposition des utilisateurs.  
  
> [!NOTE]  
>  Les administrateurs peuvent réactiver les membres supprimés. Pour plus d’informations, consultez [Réactiver un membre ou une collection &#40;Master Data Services&#41;](../reactivate-a-member-or-collection-master-data-services.md).  
  
## <a name="prerequisites"></a>Prérequis  
 Pour effectuer cette procédure :  
  
-   Vous devez avoir l’autorisation d’accéder à la zone fonctionnelle **Explorateur** .  
  
-   Vous devez disposer d'une feuille de calcul active qui contient des données managées MDS.  
  
### <a name="to-delete-a-mds-managed-row-of-data"></a>Pour supprimer une ligne de données managées MDS  
  
1.  Sélectionnez la ligne entière à supprimer en cliquant sur le titre de ligne.  
  
2.  Dans le groupe **Publier et valider** , cliquez sur **Supprimer**.  
  
3.  Dans la boîte de dialogue de confirmation, cliquez sur **Oui**.
