# <a name="properties-property-servernetworkprotocolipaddress-class"></a>Propriété Properties (classe ServerNetworkProtocolIPAddress)
  Obtient les propriétés associées à une adresse IP.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object  
.Properties [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Objet de [classe ServerNetworkProtocolIPAdress](servernetworkprotocolipaddress-class.md) qui représente une adresse IP pour le protocole réseau sur une instance de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Tableau d'objets qui représentent les propriétés prises en charge par l'adresse IP.
