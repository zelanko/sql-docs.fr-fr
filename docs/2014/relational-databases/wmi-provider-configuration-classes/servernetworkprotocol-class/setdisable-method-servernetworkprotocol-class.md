# <a name="setdisable-method-servernetworkprotocol-class"></a>Méthode SetDisable (classe ServerNetworkProtocol)
  Désactive le protocole réseau serveur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object  
.SetDisable()  
  
```  
  
## <a name="parts"></a>Éléments  
 *objet*  
 Objet [ServerNetworkProtocol Class] ServerNetworkProtocol-class.md) qui représente le protocole réseau utilisé par l’instance de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur uint32 égale à 0 si le service a été correctement modifié, égale à 1 si la demande n'est pas prise en charge ou égale à tout autre nombre pour indiquer une erreur.
