# <a name="print-query-results-visual-database-tools"></a>Imprimer les résultats d'une requête (Visual Database Tools)
  Pour imprimer les résultats de votre requête, vous pouvez copier le contenu du [volet Résultats](visual-database-tools.md) dans un autre programme Windows, un traitement de texte par exemple, puis le mettre en forme et l’imprimer.  
  
 Le Concepteur de requêtes et de vues met les résultats de la requête dans le Presse-papiers, où il sépare les colonnes par des tabulations et les lignes par un retour chariot/nouvelle ligne.  
  
### <a name="to-print-query-results"></a>Pour imprimer les résultats d'une requête  
  
1.  Sélectionnez les colonnes ou les lignes à imprimer.  
  
2.  Utilisez la commande **Copier** pour les mettre dans le Presse-papiers.  
  
3.  Allez dans le programme Windows à utiliser pour imprimer les résultats.  
  
4.  Utilisez la commande **Coller** pour y mettre le contenu du Presse-papiers.  
  
5.  Mettez en forme les résultats, puis imprimez-les.
