# <a name="define-a-new-attribute-manually"></a>Définir un nouvel attribut manuellement
  Cette procédure décrit comment créer un nouvel attribut dans une dimension à l’aide du volet **attributs** du concepteur de dimensions.  
  
### <a name="to-create-a-new-attribute-manually"></a>Pour créer un attribut manuellement  
  
1.  Dans le Concepteur de dimensions, ouvrez la dimension dans laquelle vous souhaitez créer l'attribut.  
  
2.  Assurez-vous que le volet **attributs** est au format grille.  
  
     Cliquez avec le bouton droit sur le volet **attributs** , pointez sur **afficher les attributs dans**, puis cliquez sur **grille**.  
  
3.  Faites défiler jusqu’à la dernière ligne de la grille, puis cliquez sur ** \<nouvel attribut>**.  
  
4.  Tapez le nom de l'attribut pour définir la propriété `Name`, puis appuyez sur Entrée.  
  
     Vous pouvez accepter les valeurs par défaut pour l'utilisation d'attribut, le type, la colonne clé et la colonne de nom ou sélectionner de nouvelles valeurs dans la liste déroulante.  
  
5.  Cliquez avec le bouton droit sur l’attribut, puis cliquez sur **Propriétés** pour configurer ses autres propriétés.
