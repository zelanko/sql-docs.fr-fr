# <a name="partitions-cube-designer-analysis-services---multidimensional-data"></a>Partitions (Concepteur de cube) (Analysis Services - Données multidimensionnelles)
  L'onglet **Partitions** du Concepteur de cube permet de créer et de modifier des partitions, notamment les paramètres de stockage, de mise en cache proactive et d'écriture différée, pour chaque groupe de mesures inclus dans le cube.  
  
 L'onglet **Partitions** est composé des volets suivants :  
  
## <a name="panes"></a>Volets  
 **Barre d’outils**  
 Permet d'utiliser les boutons de la barre d'outils pour effectuer les actions courantes relatives à l'onglet.  
  
 **Groupes de mesures**  
 Le volet **Groupes de mesures** permet de créer et de modifier des partitions pour chaque groupe de mesures du cube.
