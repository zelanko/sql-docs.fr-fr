# <a name="command-events-event-category"></a>Événements de commande, catégorie d'événement
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]La catégorie d’événement événements de commande comporte les classes d’événements décrites dans le tableau suivant.  
  
|Classe d'événements|ID d'événement|Description|  
|-----------------|--------------|-----------------|  
|Command Begin|15|Collecte tous les événements de début de commande depuis le début de la trace.|  
|Command End|16|Collecte tous les événements de fin de commande depuis le début de la trace.|  
  
 Pour plus d’informations sur les colonnes associées à chaque classe d’événements Command Events, consultez [Colonnes de données des événements de commande](../../analysis-services/trace-events/command-events-data-columns.md).
